How to read this graph
  • Each box is an ADR; the number is the canonical ID (also the filename prefix).
  • Hover over a box to see the ADR's title and a one-line summary (via the SVG <title> element).
  • Arrows point from a dependent ADR up to the ADR it depends on. ADR numbers increase as you read downward, so the arrows always point upward by intent.
  • Colours indicate impact tier (1 highest, 5 narrowest). A dashed outline marks a deprecated ADR (currently only 0021).
  • Some intra-tier arrows dip below the tier band to avoid overlapping with node boxes.
